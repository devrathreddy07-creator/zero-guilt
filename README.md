# ZERO GUILT — Simple Web App (Updated with fonts & images)

What's included
- index.html — main static page (uses Google Fonts).
- styles.css — off-white + chocolate-brown styling.
- app.js — client-side logic (flavours + feedback stored in localStorage).
- README.md — this file.

Quick publish steps (mobile)
1. Open the repo: https://github.com/devrathreddy07-creator/zeroguilt
2. Tap "Start coding" or use the direct new-file URL:
   https://github.com/devrathreddy07-creator/zeroguilt/new/main?filename=index.html
3. Create and commit index.html, styles.css, app.js, README.md (paste the contents).
4. Settings → Pages → Branch: main / Folder: root → Save.
5. Site will be available at:
   https://devrathreddy07-creator.github.io/zeroguilt/

Notes
- Images are loaded dynamically from Unsplash, so the published site needs internet to show them.
- Feedback is stored per visitor in localStorage. If you want shared feedback across users, I can add a backend later.
